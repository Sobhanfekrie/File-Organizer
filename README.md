# File-Organizer
A Windows-based File Organizer with Python GUI and optional C extension for fast file processing.
# File Organizer - سبحان فکری / Sobhan Fekrie

## Description / توضیح کوتاه
A Windows-based File Organizer that helps you sort your files by extension, creation date, modified date, or size.  
یک برنامه ویندوزی برای سازماندهی فایل‌ها که امکان مرتب‌سازی بر اساس پسوند، تاریخ ایجاد، تاریخ تغییر و حجم را فراهم می‌کند.

---

## Features / ویژگی‌ها
- Organize files by:
  - Extension / پسوند
  - Creation Date / تاریخ ایجاد
  - Modified Date / تاریخ تغییر
  - Size / حجم
- Bilingual interface: English & Persian / رابط کاربری دو زبانه  
- Optional C extension for faster file processing / افزودنی C برای پردازش سریع‌تر فایل‌ها  
- Simple and intuitive GUI using Python Tkinter / رابط کاربری ساده و کاربرپسند با Python Tkinter  
- Signature: "Made by Sobhan Fekrie" / نوشته گوشه برنامه: ساخته شده توسط سبحان فکری  

---

## Requirements / پیش‌نیازها
- Windows OS (tested on Windows 10/11) / ویندوز ۱۰ یا ۱۱  
- Python 3.x installed / نصب Python 3  
- Tkinter (usually comes with Python) / Tkinter معمولاً همراه Python نصب می‌شود  
- Optional: MinGW or Visual Studio to compile C extension / در صورت استفاده از C نیاز به MinGW یا Visual Studio
